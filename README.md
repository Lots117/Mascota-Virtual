##Mascota Virtual de Digimon

Una mascota virtual desarrollada en Python utilizando Pygame.

El jugador debe cuidar a su compañero digital manteniendo sus estadísticas de salud, amistad e higiene, mientras lo entrena para aumentar su poder y desbloquear diferentes rutas evolutivas.

Inspirado en los clásicos V-Pets y en la franquicia Digimon.

Características

* Sistema de salud, amistad e higiene.
* Entrenamiento para aumentar fuerza y poder.
* Disminución automática de atributos con el paso del tiempo.
* Sistema de guardado y carga de partida mediante JSON.
* Árbol evolutivo con múltiples rutas.
* Sprites animados para cada etapa evolutiva.
* Interfaz gráfica desarrollada con Pygame.
* Compatibilidad con Pygame y pygame-ce.

Árbol Evolutivo

Jyarimon
→ Gigimon

Desde aquí la evolución depende del poder obtenido mediante entrenamiento.

Rutas disponibles:

* Guilmon → Growmon → MegaloGrowmon → Dukemon → Dukemon Crimson Mode
* Veemon → Veedramon → AeroVeedramon → UlforceVeedramon
* Veemon → ExVeemon → Paildramon → Imperialdramon Dragon Mode → Imperialdramon Fighter Mode
* Evoluciones alternativas según el rendimiento de la mascota.

Tecnologías utilizadas

* Python 3.11
* Pygame
* JSON (para guardar el avance)

Instalación

Clonar el repositorio:

Instalar dependencias:

pip install -r requirements.txt

Ejecutar:

python src/main.py

Estado del proyecto

Proyecto personal completado como una práctica personal de programación orientada a objetos, interfaces gráficas y lógica de videojuegos.
